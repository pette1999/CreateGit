# CreateGit
CreateGit is a make-life-easier tool that will automate the process of creating new peojects.

## Automation workflow
1. type create + [your project name] in your terminal
2. it will create a folder add it to the desired directory
3. open the web browser and navigates to GitHub
4. automatically logs in to my account
5. creates a new repository, then adds the git remote to my local folder
6. adds a README file, does git add, and then does a commit called “initial commit” and then pushes this to master.
